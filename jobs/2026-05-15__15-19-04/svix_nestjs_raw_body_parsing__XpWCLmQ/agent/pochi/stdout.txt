### What was done

**1. Installed the Svix SDK**

npm install svix

**2. `src/main.ts` вЂ” Raw body preservation**

NestJS's automatic body parser is disabled (`bodyParser: false`), then a custom Express `json()` middleware is mounted with a `verify` callback that stores the raw `Buffer` on `req.rawBody` before it is consumed:

```ts
app.use(
  json({
    verify: (req: any, _res, buf) => {
      req.rawBody = buf;
    },
  }),
);
```

This is the critical step вЂ” Svix's `Webhook.verify()` requires the **original, unmodified bytes**, not a re-serialised JSON object.

**3. `src/app.controller.ts` вЂ” POST `/webhook` endpoint**

The handler:
- Reads `req.rawBody` (the preserved buffer)
- Extracts the three required Svix headers: `svix-id`, `svix-timestamp`, `svix-signature`
- Calls `new Webhook(secret).verify(rawBody, headers)`
- Returns **HTTP 200** on success, **HTTP 400** on any verification failure
